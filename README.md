## 🎯 核心理念 (OpenSpec)

本项目的所有规范均遵循 **OpenSpec** 思想：

1.  **意图锁定 (Lock Intent)**：通过结构化的文档（如表格、伪代码）消除自然语言的歧义。
2.  **单一事实来源 (Source of Truth)**：规范文档即是 AI Prompt 的上下文，也是团队协作的基准。
3.  **可执行性 (Executable)**：将“操作步骤”转化为“脚本逻辑”，将“设计要求”转化为“组件参数”。

## 🛠️ 使用指南

1.  **对于 AI 助手**：请将相关规范文件作为 Context 输入，要求 AI 严格遵循其中的约束生成代码或文档。
2.  **对于研发人员**：在进行设计评审（Design Review）和代码审查（Code Review）时，请以本仓库文档为验收标准。
