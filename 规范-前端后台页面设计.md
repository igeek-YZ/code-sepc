# B端控制台设计规范

本文档定义B端控制台管理后台的视觉规范、交互规范和布局规范，用于指导设计师和开发者保持界面一致性。

---

## 一、 颜色系统

### 1.1 品牌色
| 角色 | 色值 | 用途 |
| :--- | :--- | :--- |
| **主色** | `#4F46E5` | 主要按钮、链接、选中状态 |
| **主色悬停** | `#4338CA` | 按钮悬停状态 |
| **主色激活** | `#3730A3` | 按钮按下状态 |
| **主色禁用** | `#A5B4FC` | 禁用状态 |

| **警告** | `#F59E0B` | 警告提示 |
