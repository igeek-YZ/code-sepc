# 系统环境与非功能需求

## 一、 系统环境要求
- **操作系统**：Linux CentOS 7+
- **Java 版本**：JDK 1.8+
- **Node.js 版本**：v16.18.0+
- **MySQL 版本**：5.7+
- **Redis 版本**：6.0+
- **Nginx 版本**：1.16.0+

## 二、 非功能需求
- **安全性**：
    - JWT Token 有效期 2 小时。
    - 敏感数据（手机号、身份证）AES 加密存储。
    - 验证码有效期 5 分钟。
- **性能**：
    - 页面加载 ≤ 2秒。
    - API 响应 ≤ 500ms。
- **兼容性**：
    - 支持 Chrome, Firefox, Safari, Edge。
    - 适配 1024px 以上分辨率。
- **用户体验**：
    - 清晰的错误提示，简洁的操作流程。
