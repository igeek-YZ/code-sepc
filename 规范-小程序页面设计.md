# 小程序页面设计规范 (Mini Program Design Spec)

本文档定义小程序的视觉规范、交互规范和布局规范，用于指导设计师和开发者保持界面一致性。样式参考淘宝、京东等主流电商APP。

---

#### 2.1.3 登录流程规范
| 步骤 | 操作 | 说明 |
| :--- | :--- | :--- |
| **1. 微信授权** | 点击微信授权按钮 | 弹出微信手机号授权弹窗 |
| **2. 手机号授权** | 确认授权 | 获取微信绑定手机号 |
| **3. 自动登录** | 登录成功 | 跳转首页 |
| **4. 手动登录** | 点击其他登录方式 | 展开手机号+验证码登录 |
| **5. 获取验证码** | 点击获取验证码 | 60秒倒计时 |
| **6. 登录** | 点击登录/注册 | 验证后登录 |
